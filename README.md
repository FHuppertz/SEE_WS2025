# SEE Report Group 1
The report used the template for R&D project reports in the Autonomous Systems master's program at Hochschule Bonn-Rhein-Sieg, using the `IEEETran` class.

## Prerequisites

Please make sure that you have `texlive-full` installed so that you don't face any compilation errors (on Ubuntu systems, you can install it with `sudo apt install texlive-full`).

## Usage

The main file of the report is `SEE_Report_Huppertz_HW_XX.tex`; individual sections, and the appendix are included as separate files in the `subfiles` directory.

## Software and Data
All code is written in python and included in the code folder. Required libraries include numpy, matplotlib, pandas, scipy, opencv-python. All raw data is inside the data foulder.
Important Note: Most scripts need to be executed in their folders, as they use reletive file paths.
